## Features

Hotel Manager holds the following features -

* Book rooms with complete customer details
* Check room availaibility
* Check and update room status
* Call out for driver and house-keeping services
* Sort rooms based on
  - Price
  - Bedsize
  - Cleanliness
  - Availaibility
